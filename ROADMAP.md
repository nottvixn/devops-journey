# 12‑Month Roadmap — Migration Mode (1–1.5h/day)

> Theme: Backend → DevOps Engineer (Linux + AWS) with Go

## Phase 1 — Foundations (Month 1–2)
- Linux & Terminal (WSL Ubuntu)
- Git & GitHub workflow
- Go basics (CLI + REST)
- Networking basics (HTTP, TCP, DNS)

## Phase 2 — Backend Essentials (Month 3–4)
- Go REST API (Gin or chi)
- DB basics: PostgreSQL, migrations
- Testing & benchmarking
- Containerization: Docker

## Phase 3 — DevOps Core (Month 5–6)
- CI/CD (GitHub Actions)
- Linux services (systemd), logging
- Observability: metrics + tracing
- Docker Compose, Makefiles

## Phase 4 — Cloud & SRE (Month 7–9)
- AWS core: EC2, S3, IAM, VPC
- Infra as Code: Terraform
- Kubernetes basics (kind/minikube)
- Monitoring: Prometheus + Grafana

## Phase 5 — Portfolio & Jobs (Month 10–12)
- Production‑ready mini‑platform
- Docs, diagrams, runbooks
- Resume + LinkedIn + cold outbound
- AU Sponsor targeting (482): shortlist companies, role mapping, referrals
